Android build tools for Ubuntu

The Android build tools by Google

Add our android-build-tools repository:
sudo apt-add-repository ppa:maarten-fonville/android-build-tools
sudo apt-get update
* After adding our repository you can use the "Install" button to install

Install the Android build tools package:
sudo apt-get install android-build-tools-installer
